# Pirelli Tire Shop Simulator
A interactive command-line interface (CLI) banking-style inventory application written in Python. 

# Features
- Stock Management: View live tire counts and receive new shipments.
- Smart Edge-Case Validation: Prevents negative inventory inputs and blocks sales that exceed current stock levels.
- Procedural Logic: Built entirely with clean, reusable Python functions.
